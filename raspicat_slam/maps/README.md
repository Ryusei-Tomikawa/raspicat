# Raspberry Pi Cat Sample Map Data

千葉工業大学にてRaspberry Pi Catでgmappingを用いて作成した地図データです。

[rt-net/raspicat_slam](https://github.com/rt-net/raspicat_slam)を使って家庭用ゲームコントローラでRaspberry Pi Catを操作しながら地図を作成しました。
